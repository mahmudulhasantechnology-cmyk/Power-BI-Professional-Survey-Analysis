<h2>📌 Overview</h2>
<p>
This project is the final outcome of a comprehensive Power BI tutorial series, where a complete dashboard is built using <b>real survey data from data professionals</b>.  
It demonstrates the full data analytics workflow — from <b>raw data ingestion</b> to <b>interactive dashboard creation</b>.
</p>

<p>
🎯 <b>Goal:</b> Transform raw survey data into meaningful insights about the data industry and present them through a dynamic Power BI dashboard.
</p>

<hr>

<h2>🎯 Objectives</h2>
<ul>
  <li>✔ Clean and preprocess raw survey data</li>
  <li>✔ Transform data using Power Query</li>
  <li>✔ Model relationships within the dataset</li>
  <li>✔ Create calculated measures using DAX</li>
  <li>✔ Build an interactive and visually appealing dashboard</li>
</ul>

<hr>

<h2>🛠️ Tools & Techniques</h2>
<ul>
  <li><b>Power BI</b></li>
  <li><b>Power Query (Data Cleaning & Transformation)</b></li>
  <li><b>DAX (Data Analysis Expressions)</b></li>
  <li><b>Data Modeling</b></li>

</ul>

<h2>📊 Key Steps</h2>
